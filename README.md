# SEIR-Project

Web Content Scraper
A simple Python tool that takes a URL as a command-line argument and extracts key information from the webpage.

-->>>Features->
-Page Title: Extracts and prints the title of the webpage.

-Clean Body Text: Removes unwanted HTML tags (like scripts and styles) and prints the readable text content.

-Link Extraction: Finds and lists every URL/link found on the page.

-->>>Prerequisites->
You will need the following Python libraries installed:
-requests
-beautifulsoup4
